# Estudos em Python

Este repositório contém meus estudos em Python com foco em **lógica de programação**.

O objetivo é registrar minha evolução diária, praticando conceitos fundamentais
como entrada e saída de dados, estruturas de decisão e repetição.

## Estrutura
- Pastas organizadas por semanas
- Arquivos nomeados por dia e conteúdo estudado
- Exercícios simples para fixação

## Observação
Este repositório faz parte da minha jornada de aprendizado em programação.
